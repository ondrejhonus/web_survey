## Run Locally  

### Clone the project  

~~~bash  
git clone https://github.com/ondrejhonus/web_survey.git
~~~

### Go to the project directory  

~~~bash  
cd web_survey
~~~

### Install dependencies  

~~~bash  
npm install
~~~

Start the server  

~~~bash  
node .
~~~
